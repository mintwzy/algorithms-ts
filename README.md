# algorithms-ts

This project has algorithms in TypeScript with test. 

## Test

```shell
yarn install
yarn test
```

OR

```shell
docker-compose up
```

## Graphviz

The graph/tree/list are drawn using Graphviz for easy visualization. 

To generate `.dot` graph to `.png` file

```shell
dot -Tpng demo.dot -o image.png
```
